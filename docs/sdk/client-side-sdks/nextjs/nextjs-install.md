## Requirements

- Minimum Next.js version: 14.0.5
- Minimum React version: 18.2

## Installation

To install the SDK, run the following command:

### npm
```bash
    npm install @devcycle/nextjs-sdk
  ```

### yarn
  ```bash
    yarn add @devcycle/nextjs-sdk
  ```
