# Welcome to DevCycle

<img className="image-float-right" src="preview.png" alt="Your image description" width="200"/>



DevCycle is a feature flag platform built for teams of any size, helping you easily create, rollout, and cleanup feature flags without disrupting your workflow, with integrations for tools you are already using like [GitHub](integrations), [Datadog](integrations), [Snowflake](integrations) and more!

**Get started building by installing an [SDK](sdk) for your favorite language then following our [Quickstart](/introduction/quickstart) to learn how everything works.**
